# OpenClaw 前端界面美化修改教程

本教程将指导您一步步修改 OpenClaw 的界面样式，包括：背景图片、导航栏颜色、侧边栏颜色、对话气泡颜色、按钮颜色、头像等。

## 一、准备工作

1.找到需要修改的 CSS 文件

**建议**：修改前先备份原文件，方便出问题时恢复。

## 二、修改全局背景图片

找到 `.shell` 样式规则，在末尾添加以下代码：

```css
.shell {
    /* ... 原有代码保持不变 ... */
    
    /* 新增背景图片 */
    background-image: url('background.png');  /* 改成你的图片路径 */
    background-size: cover;                  /* 图片铺满整个区域 */
    background-position: center;             /* 图片居中 */
    background-repeat: no-repeat;            /* 不重复平铺 */
}
```

> 💡 **提示**：将 `background.png` 换成你自己的图片文件名，确保图片放在正确的文件夹内。

## 七、修改全局卡片颜色（影响多个区域）

找到 CSS 文件开头的 `:root` 或 `html` 中的 `--card` 变量：

```css
:root {
    --card: rgba(5, 238, 55, 0.3);  /* 绿色，30%透明度 */
    --card-foreground: #f4f4f5;
    --card-highlight: rgba(255, 255, 255, .05);
}
```

> ⚠️ **注意**：修改 `--card` 会影响多个地方的背景色，包括左侧导航栏卡片和用户输入框，请谨慎调整。

## 十、修改头像（AI 和用户）

完整替换 `.chat-avatar` 相关样式：

```css
.chat-avatar {
    width: 40px;
    height: 40px;
    border-radius: 8px;
    background: var(--panel-strong);
    display: grid;
    place-items: center;
    font-weight: 600;
    font-size: 0;           /* 隐藏默认文字 */
    flex-shrink: 0;
    align-self: flex-end;
    margin-bottom: 4px;
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
}

/* 用户头像 */
.chat-avatar.user {
    background: var(--accent-subtle);
    color: var(--accent);
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    background-image: url('user.webp');  /* 替换成你的用户头像 */
}

/* AI/助手/工具头像 */
.chat-avatar.assistant,
.chat-avatar.other,
.chat-avatar.tool {
    background: var(--secondary);
    color: var(--muted);
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    background-image: url('ai.png');     /* 替换成你的AI头像 */
}

/* 如果使用 img 标签的头像 */
img.chat-avatar {
    display: block;
    object-fit: cover;
    object-position: center;
}
```

> 💡 **提示**：将 `user.webp` 和 `ai.png` 替换成你自己的图片文件名。

## 透明度调整说明

- `0.1` = 10% 透明度（很淡）
- `0.3` = 30% 透明度（适中，推荐）
- `0.5` = 50% 透明度（半透明）
- `0.7` = 70% 透明度（较明显）
- `0.9` = 90% 透明度（接近实色）

## 常见问题

### Q：修改后没有效果？
1. 检查是否保存了文件
2. 刷新浏览器（按 Ctrl+F5 强制刷新）
3. 检查 CSS 选择器是否正确
4. 检查是否有 `!important` 被其他样式覆盖

### Q：图片不显示？
1. 确认图片路径是否正确
2. 确认图片文件名大小写是否匹配
3. 确认图片格式是否支持（jpg、png、webp、svg 等）

### Q：想改回原来的颜色？
用备份文件恢复，或者删除你添加的代码即可。
