# OpenClaw CSS 美化工具

🎨 一个简单易用的 OpenClaw 界面美化工具，无需了解 CSS 语法，图形化界面一键修改配色。

使用前建议先自己手动将文档中的css文件导入到自己的本地先实现一个替换，否则可能会出现css无法识别修改等问题

---

## 功能特点

- 🖱️ **图形化操作** - 点点鼠标即可修改，无需编写代码
- 🎨 **多种选色方式** - 支持拾色器、10种预设颜色、透明度滑块
- 🖼️ **图片支持** - 支持修改背景图片、AI头像、用户头像
- 📝 **实时预览** - 实时显示 CSS 代码修改效果
- 🔄 **安全可靠** - 支持重置功能，恢复到原始状态
- ✨ **现代界面** - 圆润的设计风格，深色主题护眼

---

## 支持修改的元素

| 分类 | 可修改项目 |
|:---|:---|
| **全局样式** | 背景图片、顶部导航栏、侧边栏、全局卡片颜色 |
| **AI回复区域** | AI回复气泡、复制按钮(悬停)、AI气泡(悬停)、文件操作框、工具卡片、工具卡片(悬停) |
| **用户区域** | 用户气泡、用户气泡(悬停)、用户头像、AI头像 |
| **输入区域** | 输入区域背景、输入框、操作按钮 |

共支持 **17个** 界面元素的修改。

## 使用方法

### 1. 准备工作

找到 OpenClaw 的 CSS 文件，路径为：

```
C:\Users\<你的用户名>\AppData\Roaming\npm\node_modules\openclaw-cn\dist\control-ui\assets\index-*.css
```

> ⚠️ **注意**：修改前建议先备份原文件。

### 2. 导入 CSS 文件

1. 点击工具顶部的 **📁 导入CSS** 按钮
2. 选择上述路径中的 CSS 文件
3. 导入成功后，左侧列表会显示所有可修改的项目

### 3. 选择要修改的元素

1. 在左侧分类列表中点击想要修改的项目
2. 右侧会显示该元素的当前配置
3. 支持修改的项目会显示颜色编辑区
4. 支持图片的项目会显示图片选择区

### 4. 修改颜色

1. **拾色器**：点击 🎨 拾色器 按钮，从颜色选择器中选择颜色
2. **预设颜色**：点击预设颜色按钮快速选择（粉色、蓝色、红色等10种颜色）
3. **透明度**：拖动透明度滑块调整透明度（0%-100%）

### 5. 修改图片

1. 点击 **📁 选择图片** 按钮
2. 选择图片文件（支持 PNG、JPG、WebP、GIF、SVG 格式）
3. 输入框中会显示选中的图片文件名

### 6. 应用并保存

1. 点击 **✅ 应用修改** 按钮应用更改
2. 点击 **💾 保存CSS** 按钮保存文件
3. 将保存的文件替换原 CSS 文件即可

### 7. 重置

如果修改不满意，点击 **🔄 重置** 按钮可恢复到导入时的原始状态。

## 透明度说明

- `0.1` = 10% 透明度（很淡）
- `0.3` = 30% 透明度（适中，推荐）
- `0.5` = 50% 透明度（半透明）
- `0.7` = 70% 透明度（较明显）
- `0.9` = 90% 透明度（接近实色）

## 常见问题

### Q：修改后没有效果？

1. 确认已保存 CSS 文件
2. 确认替换了正确的 CSS 文件
3. 刷新浏览器（按 Ctrl+F5 强制刷新）
4. 检查是否有 `!important` 被其他样式覆盖

### Q：想恢复原样式？

1. 使用工具的 **🔄 重置** 功能
2. 或用备份文件恢复
